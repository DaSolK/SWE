StudentTest.java 파일 사용법

파일을 실행파일로 실행하면 Exit를 선택하지 않는 모든 번호에 대해 메뉴를 선택하도록 출력하는 화면이 보입니다.

1. add

add 기능은 학생 한 명, 한 명을 임시 배열 st(Student Class)에 추가하는 기능입니다.

2. view

view 기능은 학생 한 명, 한 명을 저장된 파일에서 불러와 행으로 출력하는 기능입니다.

3. update

update 기능은 학생 한 명의 ID를 입력받아 그 ID가 일치하는 학생의 휴대폰번호의 내용을 변경시켜주는 기능입니다.

4. delete

delete 기능은 학생 한 명의 ID를 입력받아 그 ID가 일치하는 학생의 정보를 배열로부터 삭제하는 기능입니다.

5. save

add로 배열에 저장했던 모든 학생들의 정보를 이제 파일로 저장시키는 중요한 기능입니다. 메뉴 7번을 눌러 종료하기 이전에 해야하는 필수작업으로, save를 통해 모든 학생들의 정보가 파일에 저장됩니다.
이 때 파일을 open하지 않고 save를 했다면 새 파일에 저장이 됩니다.

6. open

파일에 저장해 두었던 기존의 학생 정보에 추가하고 싶거나, 수정 또는 삭제하고 싶을 때, 즉 이 실행파일을 실행함과 동시에 선택하는 번호입니다. 기존 파일이 없거나 새 파일을 만들고 싶을 때는 선택의 필요가 없습니다.
