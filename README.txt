현재 코드에서의 패키지 이름이 아닌 사용자의 패키지의 명으로 바꿔주어야 함.(package sowegong; -> 사용자의 패키지 명으로)

StudentTest.java 파일 사용법

파일을 실행파일로 실행하면 메뉴를 선택하는 화면이 보입니다.
각 메뉴들은 아래와 같은 기능을 합니다.

1. add

add 기능은 학생 한 명, 한 명을 임시 배열 st(Student Class)에 추가하는 기능입니다.

2. view

view 기능은 학생 한 명, 한 명을 st배열 에서 불러와 행으로 출력하는 기능입니다.

3. update

update 기능은 학생 한 명의 ID를 입력받아 st배열에 그 ID가 일치하는 학생이 있으면 새로운 휴대폰번호를 입력받고 그 학생의 휴대폰 번호를 새오운 번호로 업데이트 시켜주는 기능입니다.

4. delete

delete 기능은 학생 한 명의 ID를 입력받아 그 ID가 일치하는 학생의 정보를 st배열로부터 삭제하는 기능입니다.

5. save

그동안 st배열에서 이루어 졌던 작업을 파일로 저장시키는 중요한 기능입니다. 메뉴 7번을 눌러 종료하기 이전에 해야하는 필수작업으로, save를 통해 그동안 작업했던 모든 학생들의 정보가 파일에 저장됩니다. 작업의 마지막에 이 기능을 수행하지 않으면 작업했던 데이터가 파일로 저장되지 않습니다. 
저장 될 때 기존 파일이 없어 프로그램 시작 시 파일을 open하지 않았다면 새 파일에 저장이 됩니다. 만약 기존 파일이 있었다 해도 처음에 open 하지 않았다면 save 했을 때 기존파일은 사라지고 저장되는 내용을 덮어쓰게 됩니다. 

6. open

파일에 저장해 두었던 기존의 학생 정보에 추가하고 싶거나, 수정 또는 삭제하고 싶을 때, 즉 이 실행파일을 실행함과 동시에 선택하는 번호입니다. 이 기능을 수행하면 기존 파일에 save했던 내용을 st배열, 다시말해 프로그램 내로 불러옵니다. 기존 파일이 없거나 새 파일을 만들고 싶을 때는 선택의 필요가 없습니다. 만약 기존 파일의 내용이 필요없을 경우 open하지 않으면 작업이 끝난 후 save 시 자동으로 덮어쓰기 되서 사라집니다. 

7. Exit

프로그램을 종료하는 기능입니다.
